# Maze Pathfinding Visualizer

Interactive web-based pathfinding visualizer built using Flask, JavaScript, HTML, and CSS.

## Features

- A* Algorithm
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Maze Generation (DFS Backtracking)
- Adjustable Speed Slider
- Interactive Grid

## How to Run

1. Install dependencies
pip install -r requirements.txt


2. Run the app


python app.py


3. Open browser:


http://127.0.0.1:5000


## Controls

- First Click → Start Node
- Second Click → End Node
- Click cells → Add/Remove walls
- Press M → Generate Maze
- Use slider → Adjust speed
